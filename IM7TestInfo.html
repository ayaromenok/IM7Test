ImageMagick 7 Test
for Android
Demo application for Android_ImageMagick7 project:

 - demonstrate a benefit of OpenMP build of ImageMagick 7;
 - can be used for performance measurement;

Famous Lena image used as a source image, while DistortImage fucntion from MagickCore with BarrelDistortion method (with parameters from LG G5 wide camera) used for performance test.
